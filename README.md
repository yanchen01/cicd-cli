# Usage
<!-- usage -->
```sh-session
$ npm install -g cicd
$ cicd COMMAND
running command...
$ cicd (-v|--version|version)
cicd/0.0.0 win32-x64 node-v12.16.1
$ cicd --help [COMMAND]
USAGE
  $ cicd COMMAND
...
```
